OED Tools: NetSetMan

The problem

I manage many networks and quite often I work at customer site.

For every site I need and IP address, gateway, Wi-Fi network, sometimes proxy, a printer and many other unique network settings.

Change IP address on Windows is a process that takes too much time and many clicks.

The automation

I used to use netsh.aspx) scripts to do that but it was hard to maintain.

The OED solution: NETSETMAN

This software allows to change many network parameters, browser proxy, printer, smtp server, run custom scripts and set many other parameters.

NetSetMan has a free edition for personal usage with some feature limitations, I bought the Pro Edition it definitely worth the price.
