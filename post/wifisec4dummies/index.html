WiFi security 4 dummies

Customer site, their security policy for WiFi is an SSID with a secret PSK: the IT manager type the password on my PC to allow me to access the network.

Maybe he’s not aware of my keylogger or that the password con be easily retrieved in cleartext. Let’s see how.

Windows 10: retrieve the WiFi PSK in cleartext

Show a list of all WiFi profiles configured on the PC:

<code>netsh wlan show profiles
</code>

Show the WiFi network currently in use:

<code>netsh wlan show networks
</code>

Change PROFILENAME with the WiFi profile:

<code>netsh wlan show profiles name="PROFILENAME" key=clear
</code>

Wrap up

Never type your passwords in a PC you don’t control - keylogger exists - and don’t rely on a secret PSK for your WiFi security.
