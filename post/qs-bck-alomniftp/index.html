Quick script: backup multiple Alcatel Omniswitch configuration files

A quick batch script I did to backup configuration files of multiple Alcatel Omniswitch via FTP.

All files are named boot.cfg so the script renames each file appending the IP address.

Create a file with all the IP of switches to backup - iplist.txt

10.10.29.10
10.12.29.10
10.14.29.10
10.14.29.11
10.14.29.12
10.14.29.13
10.14.29.14
10.14.29.15

Create a FTP script to login and backup the configuration file - ftp.scr

Note: user admin password switch are defaults on Omniswitch

admin
switch
cd ..
cd working
get boot.cfg
quit

Create the actual script file - getconfig.bat

for /f "tokens=*" %%i in (iplist.txt) do (
echo %%i
ftp -s:ftp.scr %%i
ren boot.cfg boot.cfg-%%i )

Now run the script and see the magic happen.
