Quick tool: PingInfoView

_PingInfoView is a small utility that allows you to easily ping multiple host names and IP addresses, and watch the result in one table. It automatically ping to all hosts every number of seconds that you specify, and displays the number of succeed and failed pings, as well as the average ping time._

OS: Windows

License: freeware

How I use it

I find it particularly useful during network migrations to check if all the hosts are available before and after. Host ip addresses in my docs are usually in an spreadsheet file so a quick copy&paste in PingInfoView is all I need.
