OSX as Syslog server

Tuesday, January 11, 2011
osx

How to keep track of logs of a Cisco router using OSX? Easy task! Enable syslog listener on OSX: instructions here add this line to /etc/syslog.conf local7.* /var/log/cisco.log enable logging on the router: logging facility local7 ! ip address of iMac logging 10.0.0.1 install the Syslog Widget and configure it to check file /var/log/cisco.log. Use GeekTool if you prefer to view logs on the desktop.
