TAC Security Workshop

28 Apr 2017, 00:00
cisco / events

This week I had the opportunity o attend the TAC Security Workshop at Cisco Offices in Krakow.

I never attended similar events before and I was curious to see what’s the difference with PVT and other partner events.

Agenda

The agenda was simple: two days with presentations in the morning, labs in the afternoon, and one last day with just presentations.

Let’s be clear: presentations were not the usual boring slides showing all the new features of the product. All the speakers were TAC engineers showing common problems/misconfigurations and the best design and implementation practices. I learned some very useful commands and some sessions show all the troubleshooting steps a TAC engineers follows to fix issues.

Note: videos will be available soon in SalesConnect.

The event included a visit to the SOC.
